Czech Payment Integration

This module enables seamless integration of local Czech payment gateways, including GoPay and PayU Czech, into your Odoo system. It provides secure and efficient payment processing for businesses in the Czech Republic.

Key Features

  • Support for GoPay and PayU Czech APIs
  • Real-time payment processing
  • Customizable callback URLs for transaction confirmation
  • User-friendly configuration interface

How It Works

After installing the module, you can configure payment providers through the Odoo interface, allowing your customers to use their preferred local payment methods.
